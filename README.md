LanMessager is a LAN-based chat application built using C# and Windows Forms dotnet 4.5.

## Features
- Peer discovery in local network
- TCP-based messaging
- Simple Windows UI
- No internet required (works over LAN)

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- TCP Sockets

## How to Run
1. Open solution in Visual Studio
2. Build the project
3. Run on multiple PCs in the same network
